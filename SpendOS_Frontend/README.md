# 🚀 SpendOS Frontend

The frontend interface for the Smart Procurement Platform, built with **React**, **Vite**, and **Tailwind CSS**.

## 📑 Overview

SpendOS provides a premium dashboard for managing procurement analyses. It features:

- **Interactive Procurement Form**: Define requirements, budget, and custom scoring weights.
- **Real-time Analysis Tracking**: Monitor background AI workflows.
- **Rich Result Visualizations**: Deep-dive into vendor rankings, risk breakdowns, and AI-generated explanations.
- **Session History**: Access past analyses with full user isolation.

## 🛠 Setup & Installation

### 1. Install Dependencies

```bash
npm install
```

### 2. Configure Environment

Create a `.env` file in this directory:

```env
VITE_API_URL=http://localhost:8000
```

### 3. Start Development Server

```bash
npm run dev
```

## 🧪 Development Scripts

- `npm run dev`: Start the Vite dev server.
- `npm run build`: Build for production.
- `npm run lint`: Run ESLint.
- `npm run preview`: Preview the production build locally.

## 🎨 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Routing**: React Router DOM
